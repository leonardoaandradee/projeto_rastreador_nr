Monitoramento de Cargas

Este projeto consiste em um sistema de monitoramento de cargas refrigeradas ou congeladas, capaz de acompanhar em tempo real a temperatura e a localização durante o transporte da carga.

A solução utiliza tecnologias de IoT para garantir maior controle logístico e a segurança da carga.

Objetivo

O objetivo do projeto é facilitar a logística da carga sem a necessidade de verificação constante da temperatura, permitindo que o compartimento seja aberto apenas no momento necessário, evitando variações térmicas indesejadas. Além disso, o sistema possibilita o controle das rotas percorridas pela carga durante o transporte.

Tecnologias Utilizadas

  • ESP32 Wi-Fi/Bluetooth (DOIT DevKit): microcontrolador principal com suporte a bateria.
  • Módulo GPS GY-NEO6MV2: responsável pela localização em tempo real.
  • Sensor de temperatura DHT22: mede a temperatura do baú refrigerado.
  • LEDs indicadores: um LED indica o status de conexão, outro indica se o dispositivo está ligado/desligado e o LED RGB representa o nível de bateria.
  • Botão liga/desliga: controle manual do dispositivo.
  • MQTT (EMQX): envio de dados para a plataforma.
  • Dados coletados: o sistema coleta informações de temperatura e localização, permitindo acompanhar o trajeto do caminhão e a variação de temperatura ao longo do percurso.

Arquitetura do Sistema

  • Inicialização: o dispositivo O sistema utiliza um ESP32 para coleta de dados. é ligado e alimentado por bateria, iniciando o sistema de monitoramento.
  • Coleta de dados: o sensor de temperatura mede as condições internas da carga e o módulo GPS obtém a localização em tempo real.
  • Envio periódico: os dados de temperatura e localização são enviados a cada 5 minutos.
  • Comunicação MQTT (EMQX): o O sistema utiliza um ESP32 para coleta de dados. envia os dados via protocolo MQTT para o broker, responsável por intermediar a comunicação.
  • Processamento: a plataforma recebe e organiza os dados enviados pelo dispositivo.
  • Armazenamento: as informações são salvas em um banco de dados MySQL para registro e histórico.
  • Visualização: os dados são exibidos em tempo real no aplicativo e dashboard, como o IoT MQTT Panel.
  • Monitoramento: o sistema permite acompanhar o trajeto do caminhão e a variação de temperatura durante o transporte.
